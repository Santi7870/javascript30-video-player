- JavaScript30 – Día 11: Custom Video Player (Versión Mejorada)

Este proyecto es una adaptación del reto Day 11 – Custom HTML5 Video Player de JavaScript30 por Wes Bos.
El objetivo fue replicar el funcionamiento del reproductor utilizando únicamente HTML, CSS y JavaScript, además de implementar mejoras visuales al estilo de un reproductor moderno.

-  Objetivo del Reto

Construir un reproductor de video personalizado que permita:

Reproducir y pausar el video

Controlar el volumen

Ajustar la velocidad de reproducción

Adelantar y retroceder usando la barra de progreso

Mostrar controles visualmente mejorados

- Tecnologías Utilizadas

HTML5

CSS3 (diseño personalizado estilo Netflix)

JavaScript (manipulación del DOM y eventos)

- Estructura del Proyecto

video-player/
 ├─ index.html
 ├─ style.css
 ├─ app.js
 └─ assets/
